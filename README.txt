Alta Listas Comerciais — Neonatos WhatsApp v2

Arquivo principal:
- index.html

Uso pelo vendedor:
1. Abrir o index.html publicado em um link enviado no WhatsApp.
2. Buscar por produto ou código: Colostro, AltaLYTE, SOS Beef, X0092, 744637 etc.
3. Informar quantidade para o painel identificar automaticamente a faixa de preço.
4. Usar "Copiar preço" ou "WhatsApp" para enviar ao cliente.

Uso pela gerente:
- A base modelo está no Excel incluído no ZIP.
- A área "Gerente" permite testar upload de Excel no próprio navegador.
- Nesta versão estática, o upload fica salvo apenas naquele navegador. Em produção, a mesma lógica deve ser ligada a um backend ou Google Sheets para publicar a atualização para todos.

Melhorias da v2:
- Interface mobile-first para abrir dentro do WhatsApp.
- Busca e quantidade no topo.
- Identificação automática da faixa de preço por quantidade.
- Mensagens curtas para copiar/enviar pelo WhatsApp.
- Atalhos rápidos por produto/código.
- Link com parâmetros de busca (?q=AltaLYTE&qtd=25) para compartilhamento.
